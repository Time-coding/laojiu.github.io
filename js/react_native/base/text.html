# Text

# 组件引入

import Text from "..."

# 组件功能

  • 给文本添加跳转链接
  • 文本自动识别http链接，并跳转
  • 文本高亮

# Demo

  • 文本自动识别http链接，并跳转

    <Text
  content={'百度的网址是https://www.baidu.com/，还可以设置'}
/>

  • 给文本添加跳转链接

    <Text
  content={'给文本添加百度跳转链接'}
  linkUrl={'https://www.baidu.com/'}
/>

  • 文本高亮,默认#号

         <Text
      isHight
      content={'还是#适合发呆发呆发呆#'}
    />

# props

Attribute Description Type Accepted values default
style 样式
content text内容 String
color 文本颜色 String
fontSize 文本大小 Number
isHight 是否高亮 Boolean false
hightColor 文本高亮颜色
lineHeight 行高
textSymbol 分隔规则符号标示 #
linkUrl 跳转链接
onPress 点击事件
onLongPress 长按触发事件
