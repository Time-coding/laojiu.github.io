# 面试说明

面试是作为一个员工必须经历的事情，在去"求圣"的途中我们会遇到各种各样的面试官和问题。

每个公司团队不一样，团队氛围不一样，对技术的侧重点也不一样，有的可能偏业务、有的偏基础、有的偏框架、有的偏逻辑思维等等。

在众多机会中总有你的一席之地，这个是我们坚信不疑的事情。

作为面试者，我们需要在持续学习的同时，还要有眼光，你的性格和你的追求是否符合团队氛围。还需注意的一点就是态度要端正，不卑不亢，知道多少说多少，不要绕来绕去。切忌东张西望。

作为面试官，我们要在考验基础的同时，也要看品行与态度，很多时候面试官是无法在短时间内看出一个人的，但是一些小小的细节会报漏很多东西，比如东张西望、比如在问到一个比较复杂问题的时候他的回答问题的语气与态度。

# 活动说明

面试题每年是不一样的，深度和广度会随着行业的发展而改变，所以我打算搞一个活动来积累知识点。

如果你也有很多面试上的经验和困惑，可以关注我的微信公众号，留言。
